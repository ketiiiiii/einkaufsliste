# Vorgabe (Arbeitsbasis)

## Ziel
Einfache Einkauf-App.

## Kernanforderungen
- Login
- Projekt auf Githup laden, dann
- Veröffentlichung über Render
- Schlichtes, cooles, knackiges Design
- Mehrere Einkaufslisten (wie Gruppen): erstellen / bearbeiten
- Pro Liste: Positionen ohne Artikelstamm
  - Bezeichnung
  - Menge
  - Einheit (Auswahlliste)
- Pro Position Button „Erledigt“: Position wird standardmäßig ausgeblendet
- Button um ausgeblendete Positionen einzublenden
- Immer max. 10 Positionen anzeigen, Button „Nächste 10“ lädt weitere

## Offene Punkte
- „…und alles noch was ich vergessen habe“: wird iterativ ergänzt.
